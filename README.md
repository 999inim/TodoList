# TodoList
Make Simple Cooperative TodoList App with NodeJS, Browersify. 

## 개발환경
1. 서버
NodeJS

2. BuildTools
+ Nodemon
+ Watchify

3. 개발툴
+ InteliiJ IDEA
(anything is ok.)

## Installation
1. download project
프로젝트 소스코드 fork나 clone하여 로컬에 받아온다.
2. Install dependencies
```
# npm install 
```
3. make bundle file
```
# browserify ./public/Main/js -o ./public/bundle.js
```
4. Run the tests
```
# node ./app.js
```
